Detailed Description

Helper class for colored noise generation with a simple three parameter model proposed in:

"On Noise Modeling for Power Line Communications" by Luca Di Bert, ..., Andrea M. Tonello 2011 IEEE International Symposium on Power Line Communications and Its Applications

NoisePsd = a + b|f|^c [dBm/Hz]

with f in Mhz
